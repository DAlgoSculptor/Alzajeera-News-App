### 🔄 Data Flow Architecture

<br/>

```mermaid
%%{init: {'theme':'base', 'themeVariables': { 'primaryColor':'#3DDC84','primaryTextColor':'#000','primaryBorderColor':'#00C853','lineColor':'#3DDC84','secondaryColor':'#00E676','tertiaryColor':'#B2DFDB'}}}%%
sequenceDiagram
    participant UI as 📱 NewsScreen
    participant VM as 🧠 ViewModel
    participant Repo as 📦 Repository
    participant API as 🌐 NewsAPI
    
    UI->>VM: 🔄 Request News
    activate VM
    VM->>Repo: 📡 fetchNews()
    activate Repo
    Repo->>API: 🌍 GET /top-headlines
    activate API
    API-->>Repo: ✅ NewsResponse
    deactivate API
    Repo-->>VM: 📰 List<Article>
    deactivate Repo
    VM-->>UI: 🎨 Update State
    deactivate VM
    UI->>UI: ✨ Display Articles
```

**Endpoint:**
```
GET https://newsapi.org/v2/top-headlines
```

<br/>

| Parameter | Type | Description | Default |
|:---------:|:----:|:-----------:|:-------:|
| `country` | string | News country | `us` |
| `apiKey` | string | Your API key | Required |
| `pageSize` | integer | Per page | `20` |
| `page` | integer | Page number | `1` |
